Unity & C# Dust Cleanup System 🧹💨

This system spawns dirt randomly within a defined area and allows the player to clean it using a broom. The system tracks the dirt objects and handles the cleaning interaction. Most settings can be adjusted in the Inspector.

Features
- Random dirt spawning in a defined area
- Configurable spawn interval, maximum dirt count, and height offset
- Player can pick up a broom and clean dirt
- Adjustable cleaning radius and scrub speed
- Inspector-based customization
- Modular and easy to integrate into any scene

Requirements
- Unity 2020.3+
