Unity & C# Day/Night Cycle System 🌞🌙

This system implements a **dynamic day-night cycle** in Unity. The scene automatically progresses through morning, noon, afternoon, sunset, and night, updating the skybox and sun lighting accordingly.

Features

- Hour-based day-night cycle (0–24 hours)
- Configurable day length in minutes
- 5 skyboxes: Morning, Noon, Afternoon, Sunset, Night
- Sun light with dynamic color and intensity
- Smooth transitions between different times of day
- Easy to integrate and customize

How It Works

- Time progresses automatically based on the `dayLengthInMinutes`.  
- Sun rotates and changes color/intensity depending on the current time:   
- Skybox updates automatically to match the time of day.  

Requirements

- Unity 2020.3 or newer  
- Compatible with HDRP or URP pipelines
