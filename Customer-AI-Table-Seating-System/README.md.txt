Unity & C# Customer AI Table Seating System

Lightweight and modular AI system for managing customer behavior, table seating, and dynamic spawning in Unity. Built using NavMesh for movement and a flexible table-seat allocation system.

Features
- AI customers with full behavior loop (arrive → sit → eat → leave)  
- Dynamic table & seat assignment  
- Seat occupancy system  
- Continuous customer spawning  
- NavMesh-based movement  
- Automatic seat detection via hierarchy  

Modular & Extendable
- Modular: easy to integrate into any project  
- Extendable: can add new AI behaviors, table types, or queue systems  
- Works with any NavMesh setup  
- Automatic seat detection via hierarchy  

Requirements
- Unity 2020.3+  
- NavMesh system enabled
