Unity & C# Drink Shelf System

Modular item placement system built in Unity using C#.

This system is not limited to drinks; it can be used for placing and organizing any type of item.

Features
- Item pickup and placement system  
- Box-based item quantity management system with limited capacity  
- Shelf slot system for organized placement  
- Identity-based tracking for placed items  
- Reversible placement (place and retrieve items)  
- Raycast-based interaction system  
- Inspector-driven configuration  
- Modular system design with easy scene integration  

System Overview
Players place items into shelf slots and retrieve them using an ID-based system. World objects and inventory state remain synchronized.

Requirements
- Unity 2020.3+
