Locked Door System

Modular and extendable gameplay system in Unity featuring key-based doors, inventory integration, raycast interaction, and dynamic UI feedback.

Features
- Key pickup & inventory system  
- Locked / unlocked doors with key IDs  
- Raycast-based interaction (E key)  
- Interface-based system (IInteractable)  
- Smooth door animation & double door support  
- Audio feedback and UI interaction text  
- Spam protection & delayed unlock  
- Inspector-based configuration  
- Save/Load compatible (door state via SaveableEntity)   

Modular & Extendable
- Easy to integrate into any project  
- Works with custom inventory and interaction systems  
- Expandable with new interactables or mechanics  

Requirements
- Unity 2020.3+
